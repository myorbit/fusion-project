FUSION Project is an open-source framework for creating embedded sensor platforms based on a modular principle. Its core features are well integration of machine learning and sensor fusion APIs and its extremely flexible and modular design. It is planned to support several embedded hardware platforms, like the Raspberry Pi Zero.
